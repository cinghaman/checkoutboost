Microsoft Pixel Tracking for Shopify Plus

Create an Event-based Conversion Goal in Microsoft Ads

In Microsoft Ads, go to Tools > Conversion Goals.

  • Click Create Conversion Goal.
  • For Conversion Goal, select Website.
  • For Conversion Goal Type, select Purchase > then Event.
    Note: From my support requests in some cases using destination URL option worked, so if your use case is one of that try either events or destination URL which ever works for you.

    Then in your Microsoft ads conversion, change it to track the destination URL to: Contains: processing?completed=true

  • Setup Tagging > Action > Purchase and Category > Checkout
  • When do you want to track this event? Track Events on inline action

Once you are logged into your Shopify backend go to settings > Customer Events > Add Custom Pixel
